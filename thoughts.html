    WWW

    Date Published: May 31, 2022
    Author: Shivani


    I have been working on this website since 25th May. And I had decided that I'll atleast work on it for a week before making it public. All of the codes are written by me. Only a few times I took help of google, mostly in the animation part and also for some other things. But I don't wanted to add something fancy because then I'd have to copy the codes and I didn't wanted to do that. So, this website is made with purely HTML and CSS.

    Actually, HTML and CSS are quite straightforward once you get the hang of it. It's very close to english language, but not the programming languages like Python or C or Javascript. I still haven't built an intuition towards them. Like words come in your mind when you write. Similarly, the functions and the logic of the code should come to your mind as you write. It surely does happen with HTML and CSS but not so much with other languages. But I really want to be comfortable with those languages as well.
